# Rustle (如梭) - 局域网消息与文件传输工具

基于 Rust 和 egui 开发的轻量级局域网通讯工具。

## 主要功能

- **即时通讯**：支持局域网内自动发现节点，实时文本聊天。
- **可靠传输**：实现消息 ACK 确认机制，支持离线消息自动重发，确保消息不丢失、不重复。
- **文件传输**：支持单个文件及整个文件夹的传输，实时显示传输进度、速度及剩余时间。
- **历史记录**：聊天记录自动保存至系统临时目录，方便查看。
- **多语言支持**：全中文界面，支持微软雅黑字体。

## 运行平台

- Windows
- Linux
- macOS

## 技术栈

- **UI 框架**: [egui](https://github.com/emilk/egui)
- **异步运行时**: [Tokio](https://tokio.rs/)
- **序列化**: [Serde](https://serde.rs/)
- **网络**: UDP (发现/聊天), TCP (文件传输)

## 开发环境

- Rust 1.70+
